# sudoku

Sudoku solver using Java.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Installing

Go to /dist folder

Type the command

```
java -jar "Sudoku.jar"
```

Or,

```
Run the Sudoku.exe (Windows)
```
